# MMM-MySystem

## Installation
```
cd ~/MagicMirror/modules
git clone https://github.com/htilburgs/MMM-MySystem.git
cd MMM-MySystem
npm install
```

## Configuration
```
{
  module: "MMM-MySystem",
  position: "top_right",
  config: {
    showCPU: true,
    showMemory: true,
    showUptime: true,
    showDisk: true,
    showVolume: true,
    showIP: true,
    osVersion: "Trixie",
    tempUnit: "C"
  }
}
```
